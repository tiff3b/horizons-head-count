The head counter was created for an easy way to keep track of what server is next up for a table. Currently only designed for a mobile device. 
First screen you can choose a background.
Enter the servers name and click save. Once all servers are added, click done to go to the next screen.

<img width="249" height="511" alt="Screenshot 2025-12-08 132643" src="https://github.com/user-attachments/assets/10a97588-e999-4308-a72e-baefe0afc696" />

Add the head count by clicking the number buttons. After selecting save, the server with the least amount of people will be moved to the top so you know they are next up for a table.

<img width="246" height="512" alt="Screenshot 2025-12-08 132701" src="https://github.com/user-attachments/assets/451c21f7-f546-4397-be9e-908a66e32f33" />

When a server is done for the night, you may remove them from the count list by clicking the cut button. You will recieve a prompt to verify if you really want to remove the server.
